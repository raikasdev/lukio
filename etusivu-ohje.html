Kuinka asetan tämän verkkosivun selaimen etusivuksi?

Ohje Chromelle ja Firefoxille. Edgen oletussivua ei voi muuttaa.

Jos käytät Edgeä niin ihan oikeasti lataa joku oikea selain sitä Yritysportaalista.

Google Chrome

3. pistettä -> Asetukset -> Käynnistyksessä -> Avaa tietty sivu -> Lisää uusi sivu -> "https://lukio.raikas.dev"

Ulkoasu -> Laita päälle "Näytä Etusivu-painike" -> Lisää muokautettu verkko-osoite -> "https://lukio.raikas.dev"

Firefox

3. palkkia -> Asetukset -> Aloitussivu -> Aloitussivu ja uudet ikkunat -> Omat osoitteet -> "https://lukio.raikas.dev"

Verkkosivua kehittää ja ylläpitää Roni Äikäs (24b). Lähdekoodi. Anna palautetta ja ideoita

Tämä verkkosivu ei ole Jämsän lukion ylläpitämä tai hyväksymä.
